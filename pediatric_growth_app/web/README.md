# 儿童生长发育评估工具 - Web 版本

## 🌐 在线使用（推荐）

### 方式一：直接打开 HTML 文件
最简单的使用方式，无需安装任何依赖：

```bash
# 直接在浏览器中打开
open web/index.html  # macOS
xdg-open web/index.html  # Linux
start web/index.html  # Windows
```

### 方式二：使用 Python 简易服务器
```bash
cd web
python -m http.server 8080
# 然后访问 http://localhost:8080
```

### 方式三：Flask 后端服务（需要完整数据）
```bash
# 安装 Flask
pip install flask

# 启动服务
cd web
python app.py

# 访问 http://localhost:5000
```

## ✨ 功能特点

- 🎯 **三种评估模式**
  - 体重/年龄评估
  - 身高/年龄评估  
  - 体重/身高评估

- 📊 **详细指标**
  - 百分位评估 (P3, P15, P50, P85, P97)
  - Z 评分（标准差评分）
  - 临床解读建议

- 🎨 **简约界面**
  - 响应式设计，支持手机/平板/电脑
  - 清晰的可视化结果展示
  - 一键计算，快速出结果

- 👶 **适用人群**
  - 0-5 岁儿童（0-60 个月）
  - 男孩/女孩分别评估
  - 基于 WHO 2006 生长标准

## 💡 使用示例

1. 选择性别（男孩/女孩）
2. 输入年龄（岁 + 月）
3. 输入身高（cm）和体重（kg）
4. 点击"开始评估"
5. 查看三个维度的评估结果和临床解读

## 📁 文件说明

```
web/
├── index.html      # 主页面（ standalone，可直接打开）
└── app.py          # Flask 后端（可选，用于集成完整评估算法）
```

## 🔧 技术栈

- **前端**: HTML5 + CSS3 + JavaScript (原生，无框架依赖)
- **后端**: Python Flask (可选)
- **数据**: WHO 2006 儿童生长标准数据

## ⚠️ 注意事项

1. 本工具适用于 0-5 岁儿童
2. 早产儿需使用矫正年龄
3. 评估结果仅供参考，具体诊断请咨询专业医生
4. 建议连续监测生长趋势，单次测量价值有限
