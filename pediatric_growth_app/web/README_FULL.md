# 儿童生长发育与生命体征评估工具

## 📋 功能概述

这是一个专为儿科临床医生设计的综合评估工具，包含两大核心模块：

### 1. 生长发育评估模块
- **标准来源**：首都儿科研究所正常儿童生长发育曲线（0-18 岁）
- **评估维度**：
  - 体重/年龄 (Weight-for-Age)
  - 身高/年龄 (Height-for-Age)
  - 体重/身高 (Weight-for-Height)
- **评估指标**：
  - 百分位评估 (P3, P15, P50, P85, P97)
  - 状态判断 (低于正常、边缘范围、正常范围、高于正常)

### 2. 生命体征评估模块
- **标准来源**：《诸福棠实用儿科学》第 9 版及中国儿科临床指南
- **评估项目**：
  - ❤️ 心率 (次/分) - 按 8 个年龄段划分
  - 🫁 呼吸频率 (次/分) - 按 8 个年龄段划分
  - 🩸 血压 (mmHg) - 按性别和年龄段区分收缩压/舒张压上限

## 🚀 使用方式

直接用浏览器打开 `web/index_full.html` 文件即可使用。

### 心率参考值 (次/分)
| 年龄段 | 最小值 | 最大值 |
|--------|--------|--------|

### 血压参考值上限 (mmHg)
血压标准按性别分别提供收缩压和舒张压的参考上限 (P95)，用于快速筛查高血压。

## 💡 使用说明

1. **输入基本信息**：选择性别，输入年龄（岁 + 月）
2. **输入生长发育数据**：身高 (cm)、体重 (kg)
3. **输入生命体征数据**（可选）：
   - 心率（安静状态）
   - 呼吸频率（安静状态）
   - 收缩压/舒张压
4. **点击"开始综合评估"**：查看评估结果

## 🎨 界面特点

- **简约设计**：渐变紫色背景，清爽白色卡片
- **响应式布局**：适配电脑、平板、手机
- **直观展示**：
  - 绿色 = 正常范围
  - 黄色 = 边缘范围/需关注
  - 红色 = 异常范围
- **标签页切换**：生长发育三项评估可切换查看
- **卡片式生命体征**：每项生命体征独立卡片展示

## ⚠️ 注意事项

1. 本工具仅供参考，不能替代专业医疗诊断
2. 生命体征测量应在患儿安静状态下进行
3. 睡眠时心率可降低 10-20 次/分，哭闹时会升高
4. 血压精确评估需结合身高百分位，本工具提供的是快速筛查参考值
5. 如发现异常，请结合临床表现综合判断
